# 统计类接口文档

## 概述

统计类接口提供学生综合排名、数据统计等功能，主要用于学生管理系统中的数据分析和排名展示。

## 接口列表

### 1. 综合排名查询

**接口地址：** `GET /api/v1/statistics/comprehensive-ranking`

**接口描述：** 根据签到次数和活动总积分计算综合排名，返回指定范围内的排名数据。排名公式：签到次数 × 0.64 + 活动总积分

**请求参数：**

| 参数名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|--------|
| startRank | Integer | 是 | 起始排名（从1开始） | 1 |
| endRank | Integer | 是 | 结束排名 | 10 |

**请求示例：**
```
GET /api/v1/statistics/comprehensive-ranking?startRank=1&endRank=10
```

**响应参数：**

| 参数名 | 类型 | 描述 |
|--------|------|------|
| code | Integer | 响应状态码（200表示成功） |
| message | String | 响应消息 |
| data | Array | 学生综合排名信息列表 |

**data对象说明：**

| 参数名 | 类型 | 描述 |
|--------|------|------|
| ranking | Integer | 综合排名 |
| studentInfoId | Integer | 学生数据库表主键ID（非学号） |
| name | String | 学生姓名 |
| gender | String | 学生性别（男/女） |
| college | String | 学生所属学院 |
| major | String | 学生就读专业 |
| grade | Integer | 学生年级 |
| totalPoints | Long | 学生累计获得的活动总积分 |
| attendanceCount | Integer | 学生累计签到总次数 |
| levelCode | Integer | 学生等级代码 |
| comprehensiveScore | Double | 综合得分（计算公式：签到次数×0.64+活动总积分） |

**响应示例：**
```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    {
      "ranking": 1,
      "studentInfoId": 123,
      "name": "张三",
      "gender": "男",
      "college": "计算机学院",
      "major": "软件工程",
      "grade": 3,
      "totalPoints": 150,
      "attendanceCount": 80,
      "levelCode": 3,
      "comprehensiveScore": 201.2
    },
    {
      "ranking": 2,
      "studentInfoId": 124,
      "name": "李四",
      "gender": "女",
      "college": "信息工程学院",
      "major": "计算机科学与技术",
      "grade": 2,
      "totalPoints": 120,
      "attendanceCount": 75,
      "levelCode": 2,
      "comprehensiveScore": 168.0
    }
  ]
}
```

**业务规则：**
- 综合得分计算公式：签到次数 × 0.64 + 活动总积分
- 排名按照综合得分从高到低排序
- 得分相同的学生按查询顺序排列
- startRank必须大于等于1，endRank必须大于等于startRank
- 一次最多返回100条记录

**错误码：**

| 错误码 | 描述 | 解决方案 |
|--------|------|----------|
| 400 | 请求参数错误 | 检查startRank和endRank参数是否符合要求 |
| 500 | 服务器内部错误 | 联系系统管理员 |

## 安全说明

- 所有统计类接口都需要在SecurityConfig中配置公开访问权限
- 接口路径：`/api/v1/statistics/**`
- 无需身份验证即可访问
