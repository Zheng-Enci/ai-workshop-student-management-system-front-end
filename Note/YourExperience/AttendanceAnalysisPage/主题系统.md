# AttendanceAnalysisPage - 主题系统开发经验

## 主题切换实现

### 问题
页面需要实现白天/夜间模式切换功能。

### 分析
技术指令要求所有页面都有主题切换功能，通过点击AiWorkShop_icon.png图标实现。

### 方案
```vue
<img 
  src="@/assets/AiWorkShop_icon.png" 
  alt="AI坊" 
  class="logo" 
  @click="handleThemeToggle" 
  title="切换主题模式"
>
```

```javascript
import { useThemeStore } from '@/stores/theme'
const themeStore = useThemeStore()
const { toggleTheme } = themeStore
const { isDarkMode } = themeStore

const handleThemeToggle = () => {
  toggleTheme()
  setTimeout(() => {
    refreshCharts()
  }, 100)
}
```

### 效果
- ✅ 点击图标正确切换主题
- ✅ 图表在主题切换时重新渲染
- ✅ 符合技术指令要求

### 要点
- 统一通过点击AiWorkShop_icon.png图标实现主题切换
- 使用静态文字"切换主题模式"作为提示
- 主题切换时需要刷新相关组件

## CSS变量主题适配

### 问题
硬编码颜色值在主题切换时出现问题。

### 分析
需要使用CSS变量确保主题一致性。

### 方案
```css
:root {
  --text-primary: #000000;
  --text-secondary: #374151;
  --text-tertiary: #6b7280;
}

.light {
  --text-primary: #000000;
  --text-secondary: #374151;
  --text-tertiary: #6b7280;
}

.dark {
  --text-primary: #ffffff;
  --text-secondary: #e2e8f0;
  --text-tertiary: #cbd5e1;
}

html.dark {
  --text-primary: #ffffff !important;
  --text-secondary: #e2e8f0 !important;
  --text-tertiary: #cbd5e1 !important;
}
```

### 效果
- ✅ 白天模式文字显示为黑色
- ✅ 夜间模式文字显示为白色
- ✅ 主题切换优先级正确

### 要点
- 避免硬编码颜色值
- 使用CSS变量确保主题一致性
- 使用!important确保夜间模式样式优先级

## 排行榜主题适配

### 问题
排行榜文字在夜间模式下看不清。

### 分析
需要移除过强的文字阴影效果，符合极简主义要求。

### 方案
```css
/* 修复前 */
.dark .student-name {
  text-shadow: 0 1px 3px rgba(0, 0, 0, 0.9) !important;
  font-weight: 800 !important;
}

/* 修复后 */
.dark .student-name {
  font-weight: 600 !important;
}
```

### 效果
- ✅ 文字不再"亮亮的"
- ✅ 符合极简主义设计理念
- ✅ 夜间模式下文字清晰可读

### 要点
- 技术指令要求极简主义设计
- 避免过强的视觉效果
- 字体粗细要与整体设计协调
