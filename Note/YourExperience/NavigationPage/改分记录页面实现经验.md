### 1. 接口封装

在 `src/api/points.js` 中新增 `getAllAdjustRecordsByStudentInfoId` 函数：

**接口地址**: `GET /api/v1/points/all-adjust-reason-and-adjust-point-and-create-time-records-by-student-info-id`

**关键点**:
- 参数：`target-student-info-id`（学生数据库表主键ID，必填）
- 返回：包含改分理由、改分分数、创建时间的记录数组
- 错误处理：参数验证、网络错误、服务器错误等完善处理

**代码示例**:
```javascript
export const getAllAdjustRecordsByStudentInfoId = async (studentInfoId) => {
  try {
    const response = await api.get('/api/v1/points/all-adjust-reason-and-adjust-point-and-create-time-records-by-student-info-id', {
      params: {
        'target-student-info-id': studentInfoId
      }
    })
    return response.data
  } catch (error) {
    // 完善的错误处理
  }
}
```

```javascript
const sortedRecords = computed(() => {
  return [...records.value].sort((a, b) => {
    const timeA = new Date(a.createTime).getTime()
    const timeB = new Date(b.createTime).getTime()
    return timeB - timeA // 最新的在前
  })
})
```

#### 2.3 时间格式化

**实现**:
```javascript
const formatTime = (timeString) => {
  if (!timeString) return '--'
  try {
    const date = new Date(timeString)
    const year = date.getFullYear()
    const month = String(date.getMonth() + 1).padStart(2, '0')
    const day = String(date.getDate()).padStart(2, '0')
    const hours = String(date.getHours()).padStart(2, '0')
    const minutes = String(date.getMinutes()).padStart(2, '0')
    return `${year}-${month}-${day} ${hours}:${minutes}`
  } catch (error) {
    return timeString
  }
}
```

#### 2.4 数据加载逻辑

**实现**:
```javascript
const loadRecords = async () => {
  try {
    loading.value = true
    const token = localStorage.getItem('token')
    if (!token) {
      router.push('/login')
      return
    }

    // 获取学生数据库ID
    const studentIdResponse = await getStudentDatabaseTableId(token)
    if (studentIdResponse.code !== 200 || !studentIdResponse.data) {
      throw new Error('获取学生信息失败')
    }

    const studentInfoId = studentIdResponse.data

    // 获取所有改分记录
    const response = await getAllAdjustRecordsByStudentInfoId(studentInfoId)
    if (response.code === 200 && Array.isArray(response.data)) {
      records.value = response.data
    } else {
      records.value = []
    }
  } catch (error) {
    // 错误处理
  } finally {
    loading.value = false
  }
}
```

**关键点**:
- 先获取学生数据库ID（通过Token）
- 再根据学生ID获取改分记录
- Token失效时跳转登录页
- 完善的错误处理

## 技术要点

### 1. 接口调用流程

1. **获取Token**：从localStorage获取token
2. **获取学生ID**：调用 `getStudentDatabaseTableId(token)` 获取学生数据库表主键ID
3. **获取改分记录**：调用 `getAllAdjustRecordsByStudentInfoId(studentInfoId)` 获取所有改分记录
4. **数据处理**：按时间倒序排序，计算统计信息

### 2. 错误处理

**Token失效处理**:
```javascript
if (error.message.includes('Token无效') || error.message.includes('请重新登录')) {
  localStorage.removeItem('token')
  localStorage.removeItem('userInfo')
  router.push('/login')
}
```

**接口错误处理**:
- 参数验证失败：显示友好提示
- 网络错误：显示网络错误提示
- 服务器错误：显示服务器错误提示

### 3. 性能优化

- 使用 `repeat(auto-fill, minmax(320px, 1fr))` 实现响应式布局
