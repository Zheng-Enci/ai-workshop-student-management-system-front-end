## Element Plus

### 样式覆盖策略
1. **选择器优先级**: 使用Element Plus生成的类名组合
2. **样式穿透**: 使用`:deep()`穿透scoped样式
3. **全局样式**: 使用全局样式块覆盖第三方组件

```css
/* 方法1: 使用具体选择器 */
.dark .logout-btn.el-button--danger.is-plain {
  color: #000000 !important;
}

/* 方法2: 使用样式穿透 */
:deep(.dark .logout-btn.el-button--danger.is-plain) {
  color: #000000 !important;
}

/* 方法3: 使用全局样式块 */
<style>
.dark .logout-btn.el-button--danger.is-plain {
  color: #000000 !important;
}
</style>
```

## 数据计算与显示

### 数值计算最佳实践
```javascript
// 保留两位小数的计算
const calculateDailyAvgAttendance = (monthlyCount) => {
  const now = new Date()
  const currentDay = now.getDate()
  const daysInMonth = new Date(now.getFullYear(), now.getMonth() + 1, 0).getDate()
  const passedDays = Math.min(currentDay, daysInMonth)
  
  if (passedDays === 0) {
    return 0
  }
  
  return parseFloat((monthlyCount / passedDays).toFixed(2))
}
```

### 模板显示格式

### 日期计算技巧
- **获取本月天数**: `new Date(year, month + 1, 0).getDate()`
- **防止边界错误**: `Math.min(currentDay, daysInMonth)`
- **数值精度控制**: `parseFloat(value.toFixed(2))`

### 节假日计算最佳实践
```javascript
// 工作日计算（排除周末和节假日）
const calculateDailyAvgAttendance = (monthlyCount) => {
  const now = new Date()
  const currentDay = now.getDate()
  const year = now.getFullYear()
  const month = now.getMonth()
  
  const holidays = getHolidaysForMonth(year, month)
  
  let workingDays = 0
  const startOfMonth = new Date(year, month, 1)
  const endOfMonth = new Date(year, month, currentDay)
  
  for (let day = new Date(startOfMonth); day <= endOfMonth; day.setDate(day.getDate() + 1)) {
    const dayOfWeek = day.getDay()
    const isWeekend = (dayOfWeek === 0 || dayOfWeek === 6)
    const dateString = day.toISOString().split('T')[0]
    const isHoliday = holidays.includes(dateString)
    
    if (!isWeekend && !isHoliday) {
      workingDays++
    }
  }
  
  if (workingDays === 0) {
    return 0
  }
  
  return parseFloat((monthlyCount / workingDays).toFixed(2))
}

// 节假日数据管理
const getHolidaysForMonth = (year, month) => {
  const holidays = []
  
  const monthHolidays = {
    0: [`${year}-01-01`], // 元旦
    1: [`${year}-02-10`, `${year}-02-11`, `${year}-02-12`, `${year}-02-13`, `${year}-02-14`, `${year}-02-15`, `${year}-02-16`, `${year}-02-17`], // 春节
    3: [`${year}-04-05`, `${year}-04-06`, `${year}-04-07`], // 清明节
    4: [`${year}-05-01`, `${year}-05-02`, `${year}-05-03`], // 劳动节
    8: [`${year}-09-15`, `${year}-09-16`, `${year}-09-17`], // 中秋节
    9: [`${year}-10-01`, `${year}-10-02`, `${year}-10-03`, `${year}-10-04`, `${year}-10-05`, `${year}-10-06`, `${year}-10-07`] // 国庆节
  }
  
  if (monthHolidays[month]) {
    holidays.push(...monthHolidays[month])
  }
  
  return holidays
}
```

### 工作日计算技巧
- **周末判断**: `dayOfWeek === 0 || dayOfWeek === 6` (周日=0, 周六=6)
- **日期遍历**: 使用`for`循环遍历日期范围
- **日期格式化**: `day.toISOString().split('T')[0]` 获取YYYY-MM-DD格式
- **节假日管理**: 使用对象映射存储各月节假日
- **边界处理**: 检查`workingDays === 0`避免除零错误

### 用户体验优化
- 标签文字要准确描述数据含义（如"本月工作日日均签到人数"）
- 数值显示要保留适当精度（两位小数）
- 计算逻辑要考虑边界情况（月初、月末、无工作日）
