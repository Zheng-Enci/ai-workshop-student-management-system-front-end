# ProfilePage - 删除对话框新增密码输入框

## 问题描述

用户要求删除对话框相关代码，并在页面上新增一个密码输入框，用于在保存个人信息时验证身份。

## 联网学习总结（8个内容）

### 1. Element Plus 输入框组件文档
- 使用 `el-input` 组件创建密码输入框
- 设置 `type="password"` 指定输入框类型为密码
- 使用 `show-password` 属性添加显示/隐藏密码的切换按钮
- 使用 `maxlength` 限制输入长度

### 2. Element Plus 密码框示例
- 使用 `show-password` 属性创建可切换显示状态的密码输入框
- 密码输入框支持显示/隐藏密码功能，提升用户体验
- 使用 `placeholder` 提供输入提示

### 3. Vue.js 无障碍访问指南
- 为输入框添加适当的标签（`label`）
- 使用 `el-form-item` 的 `prop` 属性绑定验证规则
- 确保输入框符合无障碍访问标准

### 4. Vue.js 安全最佳实践
- 使用 `type="password"` 隐藏输入内容
- 使用 `maxlength` 限制输入长度
- 在表单验证中检查密码格式和长度

### 5. Element Plus 表单校验文档
- 在 `rules` 中添加密码验证规则
- 使用 `required` 确保密码必填
- 使用 `min` 和 `max` 限制密码长度

### 6. Vue.js 实时输入校验
- 使用 Element Plus 表单的实时验证功能
- 通过 `prop` 属性绑定验证规则
- 在用户输入时实时反馈验证结果

### 7. Element Plus 输入框尺寸调整
- 使用 `class="form-input"` 保持与其他输入框一致的样式
- 使用 `:disabled="!isEditing"` 控制输入框的禁用状态
- 保持表单布局的一致性

### 8. Element Plus 输入框图标使用
- 密码输入框使用 `show-password` 属性自动添加眼睛图标
- 图标位置由 Element Plus 自动处理
- 无需手动添加图标

```javascript
const saveProfile = async () => {
  if (isLoading.value) {
    ElMessage.warning('系统正在加载中，请稍后重试')
    return
  }
  
  if (!formRef.value) {
    ElMessage.warning('表单未初始化，请稍后重试')
    return
  }
  
  try {
    await formRef.value.validate()
  } catch (error) {
    ElMessage.warning('请先完善表单信息')
    return
  }

  isLoading.value = true
  try {
    const token = localStorage.getItem('token')
    if (!token) {
      ElMessage.error('请先登录')
      router.push('/login')
      return
    }

    const updateData = { ...formData }

    const response = await updateStudentInfo(token, updateData)
    
    if (response.code === 200) {
      ElMessage.success('个人信息更新成功')
      if (response.data && response.data.token) {
        localStorage.setItem('token', response.data.token)
      }
      originalData.value = { ...formData }
      formData.password = ''  // 保存成功后清空密码
      isEditing.value = false
    } else {
      ElMessage.error(response.message || '更新个人信息失败')
    }
  } catch (error) {
    ElMessage.error('更新个人信息失败：' + error.message)
    if (error.message.includes('Token无效') || error.message.includes('请重新登录')) {
      localStorage.removeItem('token')
      localStorage.removeItem('userInfo')
      router.push('/login')
    }
  } finally {
    isLoading.value = false
  }
}
```
