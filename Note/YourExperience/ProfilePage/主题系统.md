## 主题系统工作原理

### 全局主题状态管理
```javascript
// src/stores/theme.js
export const useThemeStore = defineStore('theme', () => {
  const isDarkMode = ref(false)
  
  const toggleTheme = () => {
    isDarkMode.value = !isDarkMode.value
    localStorage.setItem('theme', isDarkMode.value ? 'dark' : 'light')
    updateDocumentTheme()
  }
  
  const updateDocumentTheme = () => {
    const html = document.documentElement
    if (isDarkMode.value) {
      html.classList.add('dark')
      html.classList.remove('light')
    } else {
      html.classList.add('light')
      html.classList.remove('dark')
    }
  }
})
```

### CSS响应式主题切换

## 多端主题适配

### 设备端统一主题支持
所有设备端（手机端、平板端、电脑端）都使用相同的主题系统：

1. **统一的主题状态**: 所有设备端共享同一个主题状态
2. **一致的样式定义**: 使用相同的`html.dark`选择器
3. **统一的切换方式**: 都通过点击AI坊图标切换主题
4. **同步的状态保存**: 主题选择保存到localStorage

## 常见问题解决

### 1. 主题切换不生效
**问题**: 点击主题切换按钮后样式没有变化
**解决**: 检查CSS选择器是否正确，确保使用`html.dark`前缀

### 2. 部分元素没有主题样式
**问题**: 某些UI元素在夜间模式下显示异常
**解决**: 为所有UI元素添加对应的夜间模式样式

### 3. 主题状态不同步
**问题**: 不同页面的主题状态不一致
**解决**: 确保所有页面都使用同一个主题store

### 4. 主题切换闪烁
**问题**: 主题切换时出现闪烁
**解决**: 添加CSS transition实现平滑过渡
