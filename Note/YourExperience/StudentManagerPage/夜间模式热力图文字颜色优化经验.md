# StudentManagerPage - 夜间模式热力图文字颜色优化经验

## 问题描述
用户反馈夜间模式下热力图的时间文字有点看不清，需要优化夜间模式下的文字颜色对比度。

## 问题分析

### 原始配置问题
1. **文字颜色过亮**：夜间模式使用 `#fff` 白色文字
2. **对比度不够**：白色文字在深色背景上可能过于刺眼
3. **视觉疲劳**：过亮的文字容易造成视觉疲劳
4. **一致性不足**：与学生签到页的文字颜色不一致

### 学生签到页的优秀配置
学生签到页使用了更合适的文字颜色：
- **夜间模式**：使用 `#ccc` 浅灰色文字
- **白天模式**：使用 `#666` 深灰色文字
- **对比度适中**：既保证可读性又不会过于刺眼
- **视觉舒适**：符合夜间模式的设计原则

## 解决方案

### 1. 优化坐标轴文字颜色
```javascript
axisLabel: {
  color: themeStore.isDark ? '#ccc' : '#666',
  fontSize: 12
}
```

### 2. 颜色选择原理

#### 夜间模式颜色优化
- **从 `#fff` 到 `#ccc`**：降低亮度，减少刺眼感
- **对比度适中**：保证可读性的同时避免过度对比
- **视觉舒适**：符合夜间模式的设计原则
- **护眼设计**：减少蓝光刺激

#### 白天模式颜色优化
- **从 `#333` 到 `#666`**：降低对比度，更加柔和
- **层次感**：与背景形成合适的层次
- **现代感**：符合现代UI设计趋势

### 3. 设计原则

#### 可访问性设计
- **对比度要求**：满足WCAG可访问性标准
- **色盲友好**：确保色盲用户也能正常使用
- **视觉舒适**：避免过度对比造成的视觉疲劳

#### 用户体验优化
- **阅读体验**：文字清晰易读
- **视觉层次**：与背景形成合适的层次
- **主题一致**：与整体主题保持一致
