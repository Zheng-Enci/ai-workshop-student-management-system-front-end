# StudentManagerPage - 趋势图文字颜色统一优化经验

## 问题描述
用户反馈趋势图的文字也有点看不清，需要优化趋势图的文字颜色，确保与热力图保持一致。

## 问题分析

### 原始配置问题
1. **文字颜色不一致**：趋势图的xAxis和yAxis使用不同的文字颜色
2. **xAxis颜色过亮**：xAxis使用 `#fff` 白色文字，在夜间模式下过于刺眼
3. **yAxis颜色正确**：yAxis已经使用 `#ccc` 浅灰色文字
4. **整体不统一**：热力图和趋势图的文字颜色不统一

### 配置不一致问题
- **xAxis**：使用 `#fff` 白色文字（过亮）
- **yAxis**：使用 `#ccc` 浅灰色文字（正确）
- **热力图**：已经优化为 `#ccc` 浅灰色文字
- **趋势图**：部分优化，xAxis仍使用过亮颜色

## 解决方案

### 1. 统一所有文字颜色
将所有图表的文字颜色统一为：
```javascript
axisLabel: {
  color: themeStore.isDark ? '#ccc' : '#666',
  fontSize: 12
}
```

### 2. 颜色统一原则

#### 夜间模式统一
- **所有坐标轴**：统一使用 `#ccc` 浅灰色文字
- **热力图**：使用 `#ccc` 浅灰色文字
- **趋势图**：xAxis和yAxis都使用 `#ccc` 浅灰色文字
- **视觉一致性**：确保所有图表文字颜色一致

#### 白天模式统一
- **所有坐标轴**：统一使用 `#666` 深灰色文字
- **对比度适中**：既保证可读性又避免过度对比
- **现代感**：符合现代UI设计趋势

### 3. 优化范围

#### 热力图文字颜色
- **xAxis**：周一、周二、周三、周四、周五、周六、周日
- **yAxis**：上午、下午、晚上
- **visualMap**：数值范围文字

#### 趋势图文字颜色
- **xAxis**：日期标签（旋转45度）
- **yAxis**：数值标签
- **tooltip**：提示框文字
