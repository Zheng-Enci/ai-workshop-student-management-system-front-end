# DashboardPage - 人数组件布局调整

## 问题描述

用户要求将人数组件（el-input-number）放进时间选择块里面，摆在最后面。

## 实现方案

### 1. Desktop 版本布局调整

**位置**：`src/views/DashboardPage/DashboardPageDesktop.vue` 第27-57行

**修改**：
```vue
<div class="time-range-selector">
  <el-radio-group 
    v-model="selectedTimeRange" 
    @change="handleTimeRangeChange"
    size="small"
    class="time-radio-group"
  >
    <el-radio-button
      v-for="option in timeRangeOptions"
      :key="option.value"
      :label="option.value"
    >
      {{ option.label }}
    </el-radio-button>
  </el-radio-group>
  
  <div class="top-n-selector">
    <el-input-number
      v-model="selectedTopN"
      @change="handleTopNChange"
      :min="1"
      :max="100"
      :step="1"
      size="small"
      controls-position="right"
      class="top-n-input"
      placeholder="排行数量"
    />
    <span class="top-n-label">名</span>
  </div>
</div>
```

**关键修改**：
- 将 `top-n-selector` 移到 `time-range-selector` 内部
- 人数组件现在位于时间选择块的最后面

### 2. Desktop 版本样式调整

**位置**：`src/views/DashboardPage/DashboardPageDesktop.vue` 第1180-1190行

**修改**：
```css
.time-range-selector {
  display: flex;
  align-items: center;
  flex-wrap: wrap;
  gap: 12px;
}

.time-radio-group {
  display: flex;
  flex-wrap: wrap;
  gap: 8px;
}

.top-n-selector {
  display: flex;

```

**关键修改**：
- `time-range-selector` 使用 `flex` 布局，`flex-wrap: wrap` 允许换行
- `top-n-selector` 使用 `margin-left: auto` 自动推到右侧
