# 积分页面清空经验

## 概述

本文档总结了清空积分页面所有业务代码，保留基本页面结构的经验，包括代码清理、空白页面设计规范、企业级参考案例等方面的最佳实践。

## 需求背景

将积分页面清空为空白页面，删除所有业务功能，保留基本页面结构（返回按钮、主题切换图标、页面标题），为后续重新设计和实现提供干净的起点。

## 实施步骤

### 1. 代码清理

#### 1.1 删除业务代码

**桌面端页面（PointsDashboardPageDesktop.vue）**
- 删除排行榜相关代码：rankingList、rankingChart、ECharts相关
- 删除统计信息相关代码：maxPoints、totalParticipants、averagePoints、totalPoints
- 删除时间范围选择器：timeRangeOptions、selectedTimeRange、ElRadioGroup、ElRadioButton
- 删除所有API调用：getWeeklyRanking、getMonthlyRanking、getYearlyRanking、getLast7DaysRanking、getLast30DaysRanking、getTopStudentsByTimeRange、getCurrentMonthTop10Students
- 删除所有业务函数：loadRankingData、calculateStats、getRankClass、getLevelName、getLevelClass、getCurrentWeekStart、handleTimeRangeChange、initRankingChart
- 删除ECharts相关导入和初始化代码

**移动端页面（PointsDashboardPageMobile.vue）**
- 删除与桌面端相同的所有业务代码
- 保留移动端特定的布局样式

#### 1.2 删除未使用的导入

**删除的导入：**
- ECharts相关：`import * as echarts from 'echarts/core'`、`import { BarChart } from 'echarts/charts'`、`import { TooltipComponent, GridComponent, LegendComponent } from 'echarts/components'`、`import { CanvasRenderer } from 'echarts/renderers'`
- API相关：`import { getWeeklyRanking, getMonthlyRanking, ... } from '@/api/attendance'`
- Element Plus组件：`ElIcon`、`ElRadioGroup`、`ElRadioButton`
- Element Plus样式：`el-icon.css`、`el-radio-group.css`、`el-radio-button.css`
- 图标：`Trophy`、`User`、`DataAnalysis`、`TrendCharts`
- Vue生命周期：`onUnmounted`、`nextTick`（桌面端）

**保留的导入：**
- Vue Router：`useRouter`
- Theme Store：`useThemeStore`
- Element Plus组件：`ElButton`
- Element Plus样式：`el-button.css`
- 图标：`ArrowLeft`

#### 1.3 删除未使用的变量和函数

**删除的变量：**
- `rankingList`、`selectedTimeRange`、`selectedTopN`、`rankingChart`
- `maxPoints`、`totalParticipants`、`averagePoints`、`totalPoints`
- `timeRangeOptions`、`chartInstance`

**保留的函数：**
- `goBack`：返回导航功能
- `toggleTheme`：主题切换功能（从themeStore解构）

#### 1.4 样式清理

**删除的业务样式：**
- 排行榜相关：`.ranking-card`、`.ranking-list`、`.ranking-item`、`.rank-number`、`.rank-gold`、`.rank-silver`、`.rank-bronze`、`.student-info`、`.student-name`、`.student-details`、`.student-level`、`.student-points`、`.points-count`、`.points-label`
- 统计信息相关：`.stats-card`、`.stats-grid`、`.stat-item`、`.stat-icon`、`.stat-content`、`.stat-label`、`.stat-value`
- 图表相关：`.ranking-chart`
- 时间选择器相关：`.time-range-selector`、`.time-radio-group`

**保留的基本样式：**
- 容器样式：`.points-dashboard-container`、`.points-dashboard-mobile`
- Header样式：`.header`、`.header-left`、`.back-btn`、`.logo`、`.title-section`
- 内容区域样式：`.main-content`、`.content`
- 响应式样式：`@media`查询
