# 重复导入和未使用导入修复经验

## 问题描述

在积分看板页面重构过程中，遇到编译错误：

```
ERROR
[vue/compiler-sfc] Identifier 'useThemeStore' has already been declared. (29:9)

ERROR [eslint] 
PointsDashboardPageDesktop.vue
  142:9  error  Parsing error: Identifier 'useThemeStore' has already been declared. (29:9)

PointsDashboardPageMobile.vue
  89:10  error  'useUserStore' is defined but never used  no-unused-vars
  90:20  error  'ElIcon' is defined but never used        no-unused-vars
```

## 问题分析

### 错误原因
1. **重复导入**：`useThemeStore`在PointsDashboardPageDesktop.vue中被导入了两次（第117行和第142行）
2. **未使用导入**：`useUserStore`和`ElIcon`在PointsDashboardPageMobile.vue中被导入但未使用
3. **代码重构遗留**：在重构过程中，移除了个人积分相关代码，但未清理相关的导入语句

### 代码位置
- `PointsDashboardPageDesktop.vue` 第117行和第142行：重复导入`useThemeStore`
- `PointsDashboardPageMobile.vue` 第89行：未使用的`useUserStore`
- `PointsDashboardPageMobile.vue` 第90行：未使用的`ElIcon`

## 解决方案

### 1. 修复重复导入

#### 修复前
```javascript
// 第117行
import { useThemeStore } from '@/stores/theme'
import { useUserStore } from '@/stores/user'

// ... 其他导入 ...

// 第142行（重复导入）
import { useThemeStore } from '@/stores/theme'
```

#### 修复后
```javascript
// 第117行
import { useThemeStore } from '@/stores/theme'

// ... 其他导入 ...

// 移除第142行的重复导入
```

### 2. 移除未使用的导入

#### 修复前
```javascript
import { useThemeStore } from '@/stores/theme'
import { useUserStore } from '@/stores/user'  // 未使用
import { ElButton, ElIcon, ElRadioGroup, ElRadioButton } from 'element-plus'  // ElIcon未使用
```

#### 修复后
```javascript
import { useThemeStore } from '@/stores/theme'
import { ElButton, ElRadioGroup, ElRadioButton } from 'element-plus'
```
