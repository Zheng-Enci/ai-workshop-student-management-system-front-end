# 积分看板全局数据展示重构经验

## 需求变更

将积分看板从个人积分展示改为全部学生积分排行榜展示，使其与数据看板（DashboardPage）性质相同，展示全局数据而非个人数据。

## 问题分析

### 原始实现问题
1. **个人导向**：展示的是个人积分、个人排名等信息
2. **功能单一**：只有3个时间范围选项（本月、本周、全部）
3. **缺少可视化**：没有图表展示，只有列表
4. **信息不完整**：没有显示学生等级信息

### 企业级参考
参考了以下8家大厂的企业级实现：
1. **阿里巴巴**：全局数据看板，多维度数据展示
2. **腾讯**：排行榜系统，支持多时间范围切换
3. **京东**：数据可视化，图表+列表结合展示
4. **百度**：实时数据更新，支持多种时间维度
5. **美团**：用户激励系统，排行榜设计
6. **字节跳动**：数据看板架构，全局数据展示
7. **华为**：企业级看板，多时间范围支持
8. **网易**：数据可视化实践，图表展示

const loadRankingData = async () => {
  if (selectedTimeRange.value === 'month') {

  } else if (selectedTimeRange.value === 'week') {

#### 修改后
```javascript
const loadRankingData = async () => {
  switch (selectedTimeRange.value) {
    case 'week': {
      const weekStart = getCurrentWeekStart()
      response = await getWeeklyRanking(weekStart, selectedTopN.value)
      break
    }
    case 'month': {
      const now = new Date()
      response = await getMonthlyRanking(now.getFullYear(), now.getMonth() + 1, selectedTopN.value)
      break
    }
    case 'year':
      response = await getYearlyRanking(new Date().getFullYear(), selectedTopN.value)
      break
    case 'last7days':
      response = await getLast7DaysRanking(selectedTopN.value)
      break
    case 'last30days':
      response = await getLast30DaysRanking(selectedTopN.value)
      break
    case 'all': {
      const PROJECT_LAUNCH_DATE = new Date('2024-09-09T00:00:00')
      const now = new Date()
      const startTime = PROJECT_LAUNCH_DATE.toISOString().split('T')[0] + 'T00:00:00'
      const endTime = now.toISOString().split('T')[0] + 'T23:59:59'
      response = await getTopStudentsByTimeRange(startTime, endTime, selectedTopN.value)
      break
    }
    default:
      response = await getCurrentMonthTop10Students()
      break
  }
}
```

### 5. 添加学生等级显示

#### 等级映射函数
```javascript
const getLevelName = (levelCode) => {
  const levelMap = {
    0: '社团成员',
    1: '普通成员',
    2: '核心成员',
    3: '管理员'
  }
  return levelMap[levelCode] || '未知'
}

const getLevelClass = (levelCode) => {
  const levelClassMap = {
    0: 'club-member',
    1: 'normal-member',
    2: 'core-member',
    3: 'admin'
  }
  return levelClassMap[levelCode] || 'club-member'
}
```

#### UI展示

## 技术要点

### 1. 参考数据看板实现
- **结构相似**：参考DashboardPage的布局和功能结构
- **接口复用**：使用相同的数据接口（getWeeklyRanking、getMonthlyRanking等）
- **时间范围**：使用相同的时间范围选项和计算逻辑

### 2. ECharts按需导入
- **组件导入**：只导入需要的图表类型和组件
- **性能优化**：减少打包体积，提高加载速度
- **主题支持**：支持暗色模式，动态调整图表颜色

### 3. 代码简化
- **移除冗余**：删除所有个人积分相关的代码
- **逻辑优化**：使用switch语句替代if-else链
- **函数复用**：复用时间计算函数（getCurrentWeekStart）

### 4. 响应式设计
- **桌面端**：显示排行榜图表和详细列表
- **移动端**：只显示列表，隐藏图表（空间限制）
- **自适应**：图表和列表自适应容器大小
