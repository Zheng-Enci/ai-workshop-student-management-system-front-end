## 技术细节

### Element Plus 组件使用
```vue
<el-date-picker
  v-model="makeupForm.attendanceTime"
  type="datetime"
  value-format="YYYY-MM-DDTHH:mm:ss"
  :editable="false"
  :clearable="false"
  inputmode="none"
  popper-class="makeup-attendance-date-picker-mobile"
/>
```

### 输入法禁用策略
1. HTML 属性：inputmode="none"
2. CSS 样式：pointer-events: none
3. JavaScript：动态设置 readonly 和 pointer-events

### 确认提示框逻辑
```javascript
let confirmMessage = `确定要为学生 ${student.name} 补卡吗？\n补卡时间：${selectedTime.toLocaleString('zh-CN')}`
if (!isValidTime) {
  confirmMessage += '\n\n⚠️ 警告：补卡时间不在有效签到时间段内'
}
await ElMessageBox.confirm(confirmMessage, '确认补卡', {...})
```
