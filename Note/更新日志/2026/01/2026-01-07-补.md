## 概述
本次更新优化了学生管理页面夜间模式下的文字显示效果。

## 详细更新内容

### 1. 优化学生管理页面夜间模式文字颜色
**文件：** `src/views/StudentManagerPage/css/desktop/StudentManagerPage-StudentCards.css`

**问题描述：**
- 夜间模式下学生卡片的文字颜色 `#5a6c7d` 过于昏暗
- 在深色背景下可读性较差

**解决方案：**
- 将文字颜色调整为 `#c8c8c8`（较亮的灰色）
- 提高夜间模式下的文字可读性

**具体修改：**
```css
/* 修改前 */
html.dark .student-cards-list-item * {
    color: #5a6c7d !important;
}

/* 修改后 */
html.dark .student-cards-list-item * {
    color: #c8c8c8 !important;
}
```

## 影响范围

### 用户体验改进
- 改善了夜间模式下的视觉体验
- 提高了学生管理页面在夜间模式下的文字可读性
- 符合无障碍设计标准

---

## 注意事项

1. **颜色对比度**：新的颜色值 `#c8c8c8` 在深色背景（如 `#1e293b`）下提供了更好的对比度
2. **全局样式**：使用 `*` 选择器和 `!important` 确保所有子元素都继承该颜色
