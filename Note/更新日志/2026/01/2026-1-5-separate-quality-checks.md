# 更新日志 - 2026年1月5日（代码质量检查分离）

## 📋 概述

将代码质量检查从开发服务器启动流程中分离，通过独立命令运行，提升开发服务器启动速度。

## 🔧 修改内容

**文件**: `vite.config.js`
- 使用环境变量 `ENABLE_CODE_QUALITY_CHECKS` 控制代码质量检查插件的启用
- 默认情况下（`npm run dev`）不执行任何检查，快速启动
- 仅在环境变量为 `true` 时启用所有检查插件

**文件**: `package.json`
- 新增命令 `check:quality`：运行所有代码质量检查
- `dev` 命令保持不变，默认不执行检查

## 📝 使用方式

- `npm run dev` → 快速启动开发服务器（不执行检查）
- `npm run check:quality` → 启动开发服务器并执行所有代码质量检查
