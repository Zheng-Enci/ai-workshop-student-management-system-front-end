## 统一规范

### 加载蒙版使用规范
所有设备检测页面现在遵循统一的加载蒙版使用模式：

1. **导入依赖**:
   ```javascript
   import { useLoadingMaskStore } from '@/stores/loading'
   import LoadingMask from '@/components/LoadingMask.vue'
   ```

2. **初始化实例**:
   ```javascript
   const loadingMaskStore = useLoadingMaskStore()
   ```

3. **显示加载蒙版**:
   ```javascript
   loadingMaskStore.showLoadingMask('正在检测设备类型...')
   ```

4. **隐藏加载蒙版**:
   ```javascript
   loadingMaskStore.hideLoadingMask()
   ```

5. **模板中添加组件**:
   ```html
   <div class="{页面名}-page-device-detection-container">
       <LoadingMask/>
   </div>
   ```

### 样式命名规范
所有设备检测页面的容器样式类名格式为：
```
{页面名}-page-device-detection-container
```

示例：
- `admin-page-device-detection-container`
- `student-manager-page-device-detection-container`
- `score-change-records-page-device-detection-container`
- `profile-page-device-detection-container`
- `points-dashboard-page-device-detection-container`
- `navigation-page-device-detection-container`
- `attendance-page-device-detection-container`
- `all-members-page-device-detection-container`

### 统一的渐变背景
所有设备检测页面使用统一的渐变背景：
```css
background: linear-gradient(135deg,
```
