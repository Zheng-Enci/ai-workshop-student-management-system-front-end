# 清理邀请码相关代码

## 日期
2026-03-23

## 功能概述
清理不再使用的邀请码相关代码和样式。

## 主要改动

### 1. 删除不再使用的样式文件
- 文件：`src/views/NavigationPage/css/InvitationCode.css`
- 原因：
  - 该文件中定义的 CSS 类名在当前的 Vue 文件中都没有被使用
  - 之前的实现使用了这些样式，但现在已经改用了表单组件的方式
  - 桌面端和移动端的 `InvitationCodeForm.vue` 都使用了自己的 scoped 样式

### 2. 清理暗色模式下退出按钮样式
- 文件：`src/views/NavigationPage/NavigationPageDesktop.vue`
- 文件：`src/views/NavigationPage/NavigationPageMobile.vue`
- 删除了空的 `<style>` 标签，这些标签原本用于定义暗色模式下退出按钮的样式
- 这些样式已经不再需要

## 技术细节

### 样式管理优化
- 移除了冗余的 CSS 文件
- 清理了空的样式标签
- 所有邀请码相关样式现在都封装在各自的组件中

### 代码清理
- 删除了未使用的 CSS 类定义
- 删除了空的样式块
- 提高了代码的可维护性
