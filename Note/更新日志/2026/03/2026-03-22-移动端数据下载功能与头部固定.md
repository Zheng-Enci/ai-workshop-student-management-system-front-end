# 2026-03-22 移动端数据下载功能与头部固定实现

## 功能实现
- 新增手机端 `DataDownloadForm.vue` 组件，实现数据下载表单
- 在 `AdminPageMobile.vue` 中添加数据下载按钮切换功能
- 实现页面头部和搜索区域固定在顶部
- 添加箭头按钮，支持在退出登录和导出数据按钮之间切换

## 技术细节

### 数据下载功能
- 实现了获取上周七天日期的逻辑
- 循环调用考勤 API 获取每日签到人次
- 按索引顺序存储数据，索引 0 代表周一
- 生成 CSV 文件，第一行为 "一" 到 "七"，第二行为对应签到次数
- 集成了 CSV 导出功能，支持文件下载
- 支持下载上周签到活跃学生数据（签到次数大于3次）

### 头部固定功能
- 使用 `position: fixed` 固定头部在页面顶部
- 设置 `z-index: 1000` 确保在最上层
- 添加背景色和阴影效果，提升视觉层次
- 搜索区域固定在头部下方（top: 64px）
- 学生卡片区域添加 `padding-top: 184px` 为固定区域留出空间

### 按钮切换功能
- 添加 `showDownloadButton` 状态变量控制按钮显示
- 使用 `v-if` 条件渲染实现按钮切换
- 箭头按钮根据状态旋转（向右或向左180度）
- 箭头按钮样式优化：padding: 4px，min-width: 20px

### 移动端适配
- 按钮垂直排列，更适合手机屏幕操作
- 按钮高度设置为 48px，提升触摸体验
- 添加小屏幕适配（375px以下）：
  - 调整按钮高度为 44px
  - 调整字体大小为 14px
  - 调整标题字体大小
- 添加超小屏幕适配（320px以下）：
  - 进一步缩小按钮尺寸
  - 进一步缩小字体大小
