# 修复本周签到概览周日计算错误

## 问题描述
当本周为周日时，本周签到概览显示全部为未签到，但实际已有签到记录。

## 原因分析
代码中使用 `currentDate.getDay()` 计算本周周一日期时，周日的 `getDay()` 返回 0，而代码直接使用 `+1` 计算导致计算出的是下一周的周一。

## 修复方案
将 `getDay()` 为 0（周日）的情况特殊处理为 7，确保正确计算本周周一日期。

## 修改文件
- `src/views/AttendancePage/AttendancePageMobile.vue`

## 修改内容
```typescript
// 修改前
currentDate.setDate(currentDate.getDate() - currentDate.getDay() + 1 + index)

// 修改后
const dayOfWeek = currentDate.getDay() || 7
currentDate.setDate(currentDate.getDate() - dayOfWeek + 1 + index)
```
