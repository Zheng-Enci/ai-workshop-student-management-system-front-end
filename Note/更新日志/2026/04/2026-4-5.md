# 更新日志 - 2026年4月5日

## 📋 概述

本次更新为开发模式添加了跳过代码质量检查的选项，提升开发启动速度。

---

## 🔧 功能优化

### ✨ 新增 SKIP_CHECKS 环境变量支持

- **文件**: `vite.config.js`
- **修改**: 添加 `SKIP_CHECKS` 环境变量判断逻辑
- **说明**: 当设置 `SKIP_CHECKS=true` 时，Vite 启动时会跳过所有代码质量检查插件（ESLint、Stylelint、CSS分析、依赖检查、安全审计、注释覆盖率检查），大幅提升开发服务器启动速度

### 📝 使用方法

**跳过检查启动（Windows PowerShell）**:
```powershell
$env:SKIP_CHECKS="true"; npm run dev
```

**跳过检查启动（CMD）**:
```batch
set SKIP_CHECKS=true&& npm run dev
```

**正常启动（包括检查）**:
```bash
npm run dev
```
