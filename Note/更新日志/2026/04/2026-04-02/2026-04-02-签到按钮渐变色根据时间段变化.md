# 2026-04-02 签到按钮渐变色根据时间段变化

## 修改内容

### 1. 更新 attendance-mobile-sign-button.css

**新增时间段渐变样式**:

```css
/* 上午签到按钮 - 暖橙色渐变 */
.attendance-mobile-sign-button-button-mobile.morning {
	background: linear-gradient(135deg, #FF9800 0%, #FF5722 100%);
	box-shadow: 0 8px 28px rgba(255, 152, 0, 0.4), ...;
}

/* 下午签到按钮 - 天蓝色渐变 */
.attendance-mobile-sign-button-button-mobile.afternoon {
	background: linear-gradient(135deg, #2196F3 0%, #00BCD4 100%);
	box-shadow: 0 8px 28px rgba(33, 150, 243, 0.4), ...;
}

/* 晚上签到按钮 - 蓝紫色渐变 */
.attendance-mobile-sign-button-button-mobile.evening {
	background: linear-gradient(135deg, #673AB7 0%, #3F51B5 100%);
	box-shadow: 0 8px 28px rgba(103, 58, 183, 0.4), ...;
}
```

**更新hover效果** - 根据时间段显示不同阴影：

```css
.attendance-mobile-sign-button-button-mobile.morning:hover:not(:disabled) {
	box-shadow: 0 12px 36px rgba(255, 152, 0, 0.5), ...;
}

.attendance-mobile-sign-button-button-mobile.afternoon:hover:not(:disabled) {
	box-shadow: 0 12px 36px rgba(33, 150, 243, 0.5), ...;
}

.attendance-mobile-sign-button-button-mobile.evening:hover:not(:disabled) {
	box-shadow: 0 12px 36px rgba(103, 58, 183, 0.5), ...;
}
```

### 2. 更新 AttendancePageMobile.vue

**按钮动态class绑定**:
```vue
:class="{
	'loading': loading,
	'disabled': !isInSignTime || isCurrentSlotSigned(),
	'success': isInSignTime && isCurrentSlotSigned(),
	'morning': getCurrentTimeSlot() === 'morning',
	'afternoon': getCurrentTimeSlot() === 'afternoon',
	'evening': getCurrentTimeSlot() === 'evening'
}"
```

## 效果说明

- **上午签到时段**：按钮显示橙色渐变，与上午图标背景色一致
- **下午签到时段**：按钮显示蓝色渐变，与下午图标背景色一致
- **晚上签到时段**：按钮显示蓝紫色渐变，与晚上图标背景色一致
- **签到成功**：统一变为绿色渐变
