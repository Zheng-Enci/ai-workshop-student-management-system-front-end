# 2026-04-02 签到按钮背景渐变动画

## 修改内容

### 1. 更新 attendance-mobile-sign-button.css

**原背景样式**:
```css
.attendance-mobile-sign-button-background-mobile {
	background: linear-gradient(45deg, rgba(255, 255, 255, 0.1) 0%, rgba(255, 255, 255, 0.05) 100%);
}
```

**新背景样式（带渐变流动动画）**:
```css
.attendance-mobile-sign-button-background-mobile {
	background: linear-gradient(
		45deg,
		rgba(255, 255, 255, 0.2) 0%,
		rgba(255, 255, 255, 0.1) 25%,
		rgba(255, 255, 255, 0.2) 50%,
		rgba(255, 255, 255, 0.1) 75%,
		rgba(255, 255, 255, 0.2) 100%
	);
	background-size: 200% 200%;
	animation: attendance-mobile-sign-button-gradient-flow 3s ease infinite;
}
```

**签到成功后的背景**:
```css
.attendance-mobile-sign-button-button-mobile.success .attendance-mobile-sign-button-background-mobile {
	background: linear-gradient(45deg, rgba(76, 175, 80, 0.3) 0%, rgba(76, 175, 80, 0.1) 100%);
	animation: none;  /* 停止动画 */
}
```

**新增动画**:
```css
@keyframes attendance-mobile-sign-button-gradient-flow {
	0% {
		background-position: 0% 50%;
	}
	50% {
		background-position: 100% 50%;
	}
	100% {
		background-position: 0% 50%;
	}
}
```

## 效果说明

- **签到前/非签到时段**：背景渐变持续流动动画，吸引用户注意
- **签到成功（success状态）**：动画停止，显示绿色渐变背景，表示已完成
