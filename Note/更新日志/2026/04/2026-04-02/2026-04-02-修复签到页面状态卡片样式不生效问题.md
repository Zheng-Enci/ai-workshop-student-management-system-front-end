# 2026-04-02 修复签到页面状态卡片样式不生效问题

## 问题描述

签到页面移动端的状态卡片样式没有生效。

## 问题原因

`@import` 语句在 `<style scoped>` 块中的位置不正确。

在 Vue 的 scoped 样式中，`@import` 语句必须放在 `<style scoped>` 之后的第一行，不能放在其他样式规则之后。

## 修改内容

### 文件: `src/views/AttendancePage/AttendancePageMobile.vue`

**修改前**:
```vue
<style scoped>
/** 根容器样式：移动端全屏布局，渐变背景，相对定位 */
.attendance-container-mobile {
	...
}

... 其他样式规则 ...

@import './css/mobile/attendance-mobile-status-cards.css';

/** 桌面端提示卡片：玻璃态，水平布局，图标+文本 */
.desktop-tip-card-mobile {
	...
}
</style>
```

**修改后**:
```vue
@import './css/mobile/attendance-mobile-status-cards.css';

<style scoped>
/** 根容器样式：移动端全屏布局，渐变背景，相对定位 */
.attendance-container-mobile {
	...
}

... 其他样式规则 ...

/** 桌面端提示卡片：玻璃态，水平布局，图标+文本 */
.desktop-tip-card-mobile {
	...
}
</style>
```

## 技术细节

### Vue Scoped 样式规则

1. **`@import` 位置**: 必须在 `<style scoped>` 之后的第一行
2. **作用域**: `@import` 引入的样式也会被 scoped 作用域包裹
3. **顺序**: `@import` → 其他样式规则
