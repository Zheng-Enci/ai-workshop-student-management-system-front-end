# 2026-04-02 签到按钮渐变动画改为单向流动

## 修改内容

### 1. 更新 attendance-mobile-sign-button.css

**修改前（来回流动）**:
```css
@keyframes attendance-mobile-sign-button-gradient-flow {
	0% {
		background-position: 0% 50%;
	}
	50% {
		background-position: 100% 50%;
	}
	100% {
		background-position: 0% 50%;
	}
}
```

**修改后（单向流动）**:
```css
@keyframes attendance-mobile-sign-button-gradient-flow {
	0% {
		background-position: 0% 0%;
	}
	100% {
		background-position: 100% 100%;
	}
}
```

## 效果说明

- 渐变现在只朝一个方向（从左上到右下）流动
- 不再来回往复
- 视觉效果更流畅、自然
