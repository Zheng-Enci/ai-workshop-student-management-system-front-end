## 1. 修复 AdminPageDesktop 导入错误

### 问题描述
- 页面加载时出现错误：`getAvatarUrl is not a function`
- 导致超级管理员桌面端页面无法正常加载

### 根本原因
- AdminPageDesktop.vue:80 使用了错误的导入语法
- 错误写法：`const { getAvatarUrl } = '@/api/student'`
- 这种解构导入方式在 ES6 中是无效的

### 解决方案
- 修改为正确的 ES6 import 语句：
  ```javascript
  import { getAvatarUrl } from '@/api/student'
  ```

### 文件变更
- src/views/AdminPage/AdminPageDesktop.vue (line 80)

## 影响范围
- 超级管理员桌面端页面加载

## 测试要点
- [x] 超级管理员桌面端页面正常加载
- [x] 管理员头像正常显示
- [x] 无控制台错误
